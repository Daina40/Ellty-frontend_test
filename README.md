Technologies used:

  Backend: Django 4.x, Django REST Framework
  Frontend: HTML, CSS, JavaScript
  IDE: Visual Studio Code

Features:

  Pixel-perfect design replication from Figma.
  Fully functional interactive elements (buttons, checkboxes, etc.).
  Clean, modular, and easy-to-read code.
  Ready for API integration (using Django REST Framework). 

Setup Instructions

  1. Clone the repository
     git clone https://github.com/Daina40/Ellty-frontend_test.git
  2. Create a virtual environment
     python -m venv venv
  3. Activate the virtual environment
      Windows:
         venv\Scripts\activate
      macOS/Linux:
         source venv/bin/activate
  4. Install dependencies:
       pip install -r requirements.txt
  5. Apply migrations:
       python manage.py migrate
  6. Run the development server:
       python manage.py runserver
